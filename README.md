# Chart.Annotation.js

An annotation plugin for Chart.js >= 2.1.3


## Configuration

To configure the annotations plugin, you can simply add new config options to your chart config.

```javascript
{
	annotation: {
		annotations: [{
			type: 'line',
			mode: 'horizontal',
			scaleID: 'y-axis-0',
			value: '25',
			borderColor: 'red',
			borderWidth: 2
		}]
	}
}
```
